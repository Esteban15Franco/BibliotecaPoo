# BibliotecaPoo
Aplicación en Java para la gestión de una pequeña biblioteca, desarrollada aplicando Programación Orientada a Objetos (POO).

El sistema permite registrar libros y clientes, así como controlar
el préstamo de libros bajo
la siguiente regla principal:

**Un cliente solo puede tener un libro prestado a la vez.**
No puede llevar otro hasta que devuelva el actual.

## Objetivo

Practicar los conceptos básicos de POO como clases, objetos,
encapsulamiento y relaciones entre clases
